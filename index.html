О себе

Студент 2-го курса MIREA — Russian Technological University по направлению 'Технологии искусственного интеллекта и анализ данных'. За год работы в графическом дизайне создал более 20 проектов для малого и среднего бизнеса, включая логотипы, брендинг и веб-дизайн. Профессионально владею Adobe Photoshop, на среднем уровне Adobe Illustrator. Параллельно развиваюсь как аналитик данных, применяя математический подход и внимание к деталям, полученные в дизайне. Работаю с Python, SQL и инструментами визуализации данных. Мой уникальный опыт в дизайне позволяет создавать не только точные, но и визуально привлекательные отчеты и дашборды. Стремлюсь решать бизнес-задачи через данные, превращая сложную информацию в понятные инсайты и actionable рекомендации.

Пет-проекты по аналитике данных

01

Анализ данных о недвижимости Москвы

Исследовал открытый датасет с ценами на квартиры в Москве (15,000+ объявлений). Провел исследовательский анализ данных, выявил факторы влияющие на цену, построил корреляционную матрицу и создал визуализации в Python. Использовал данные с популярных сайтов недвижимости.

Выявил 8 ключевых факторов ценообразования
02

Дашборд продаж интернет-магазина

Создал интерактивный дашборд на основе учебного датасета Superstore Sales. Проанализировал данные за 2 года, построил KPI метрики, создал фильтры по регионам и категориям товаров. Дашборд показывает динамику продаж, топ-товары и прибыльность по сегментам.

Интерактивный дашборд с 12 визуализациями
03

Анализ данных GitHub репозиториев

Собрал данные о популярных Python репозиториях через GitHub API (1000+ репозиториев). Исследовал зависимость между количеством звезд, форков и используемыми технологиями. Создал автоматический отчет с визуализациями трендов в Python экосистеме.
